# flutter-npm-search

## 前言
这是一个使用Flutter框架开发的应用，主要功能是搜索npm上的package，显示package的信息，有4个页面，首页，搜索页， 详情页以及一个用来打开网页的WebView页面，接口来自于[npms.io](https://api-docs.npms.io/)。
